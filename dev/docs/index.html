Yama.create_surface — Method
function create_surface(
    model, dataloader::DataLoader, evaluation_step, args::SurfaceArgs,
)

Create loss surface.

Parameters

  • model: Model to use in loss function.
  • dataloader::DataLoader: Dataset on which to evaluate loss function.
  • evaluation_step: Custom-defined function which given model, mini-batch and args, computes loss on that mini-batch.
  • args::SurfaceArgs: Parameters used when computing surface.
Note

If you specified use_gpu in args, then it is up to you, to transfer mini-batch in evaluation_step function to gpu.
